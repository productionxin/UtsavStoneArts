# Utsav Stone Art — Experience Website

Built by Production X · productionx.in

## Tech Stack
- React + Vite
- React Three Fiber (R3F) + Three.js
- @react-three/drei + postprocessing
- Framer Motion
- GSAP

## Deploy to Vercel
This repo is linked to Vercel. Every push to main auto-deploys.

## Local Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
